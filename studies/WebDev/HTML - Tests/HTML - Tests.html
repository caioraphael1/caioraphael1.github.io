iframe

  • ![[teste.excalidraw]]

    • Doesn't work anywhere.

  • ![[assets/teste.excalidraw]]

    • Doesn't work anywhere.

  • ![](assets/teste.excalidraw)

    • Doesn't work anywhere.

  • <img src="assets/teste.excalidraw" >

    • Doesn't work anywhere.

  • <iframe class="excalidraw" src="assets/teste.excalidraw" ></iframe> .

    • Works on the web, but only shows the code, not the image.

    • Doesn't work elsewhere.

    • When you point an <iframe>  at it the browser loads the file as text and shows the file contents because there is no webpage or JS app to render that JSON.

  • .   - Works everywhere.   - It loads the Excalidraw application. The app is an HTML+JS page that reads the URL hash on the client and renders the JSON as a drawing. The hash fragment is never sent to the server. The app does the rendering in the browser, so the iframe shows the rendered canvas.

  • For excalidraw, I need to export as SVG or PNG, or use the excalidraw site.
