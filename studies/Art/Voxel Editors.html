VoxEdit

  • VoxEdit .

  • The app is still in beta, so the beta is free. Not sure if it will be paid in the future.

  • Besides model creation, it provides tools for rigging and animation.

  • Demonstrations {Playlist} .

  • Features:

    • The max size is 256x256 voxels (16x16x16 Minecraft blocks).

  • Impressions  [2024/07/21]

    • The "Modeler" has no advantage compared to any other software.

    • Found it suuuper laggy when using any modeling tool.

    • Files are saved in .vxm, which is annoying for compatibility.

    • When trying to save a simple model file, the app crashed on me xDDDDD.

    • The company is extremely childish and does countless ads for NFTs and "the Sandbox game world"; extremely mercenary.

    • I found the app quite bad and unusable considering how laggy the tools are, plus the crashes; all of that without any advantage over other apps.
