Symlink

  • Explanation .

Hard Link pointing to a directory

  • mklink /J "new_folder_created" "source_folder_to_link"

  • Example :

    • cd C:\blablabla\game_server

    • mklink /J "worlds" "..\client_side\worlds"

    • This makes the game server have a link to the client_side ; a new worlds  folder is created in the game server.
