TMux (Terminal Multiplexer)

  • OSs :

    • ~Windows (no native support).

    • Linux.

    • MacOS.

    • BSDs.

    • Solaris.

  • Explanation .

  • TMux with NeoVim .

    • Haven‚Äôt watched the video.

  • A powerful tool allowing management of multiple terminal sessions in a single window.

  • Sessions : A group of terminals that can persist even after closing the terminal.

  • Windows : Within a session, you can open multiple windows.

  • Panes : Each window can be split into multiple panes.

  • Weird interaction between TMux and Nvim .

    • Allows use of c-[hjkl]  to navigate BETWEEN Tmux and Nvim windows.

    • Kind of weird but seems functional.

WSL (Windows Subsystem for Linux)

  • Allows running Linux distributions inside Windows without a VM.

  • Supports Bash, Zsh, Fish, and any other Linux shell .

  • Useful for developers needing a Linux environment on Windows.

  • To install WSL and a distribution like Ubuntu: 

    wsl --install

  • Then, open Ubuntu in the Start menu.
