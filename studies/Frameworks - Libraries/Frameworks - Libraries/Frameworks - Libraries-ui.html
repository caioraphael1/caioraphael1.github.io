Render

  • Support :

    • (2025-06-25)

    • dx11, dx12, glfw, metal, opengl3, osx, sdl2, vulkan, wgpu, win32

      • Binding with support.

    • SDL3:

      • Binding without support.

    • Not sure about the reliability of any of the renderers for the binding.

  • There is a file with initialization context implementations for each render framework.

    • For example, if using GLFW, there will be:

Nuklear

  • Nuklear .

  • Nuklear-odin .

  • In C89.

  • A minimal, immediate-mode GUI library written in C that can be integrated into graphics frameworks like RayLib.

  • Nuklear includes its own simple font baking (bitmap atlas) and text-layout routines, so you can render UI text without pulling in heavy font pipelines.

  • Smaller alternative to Dear ImGUI.
